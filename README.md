# Code Server FastAPI

## Setup env and run

install anaconda

### *open Anaconda Prompt*

conda create --name envPBL5 python=3.10.10

conda activate envPBL5

pip install -r requirements.txt

### *open project*

run cmd: uvicorn main:app --reload
